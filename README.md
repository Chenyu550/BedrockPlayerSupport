# BedrockPlayerSupport

------
## 介绍

让插件对基岩版玩家更加友好  
  
功能:  
 1. 当基岩版玩家收到 tpa/tpahere 申请时会弹出GUI让玩家选择 同意/拒绝/忽略 (CMI/EssentialsX)
 2. 基岩版玩家进服注册/登录时使用GUI操作或直接跳过基岩版玩家登录环节 (AuthMe)

------

## 使用方法

直接丢进 plugins 文件夹即可  
前置:  
    - [floodgate][2] (必要)  
    - [CMI][1] [EssentialsX][3] [AuthMe][4] (可选)
