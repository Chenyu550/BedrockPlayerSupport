# 介绍
**让其他插件对基岩版玩家更加友好**  
  
BedrockPlayerSupport 是一个优化基岩版玩家体验的插件，给服务器添加了基岩版 TPA/TPAHERE GUI，给主流的基础插件(CMI, EssentialX)的 TPA/TPAHERE 功能增加了基岩版GUI  
  
给部分主流登录插件(AuthMe/CatSeedLogin/NexAuth)单独添加了基岩版玩家自动登录功能，其他登录插件也可以在配置文件中设置强制登录命令实现上述功能
