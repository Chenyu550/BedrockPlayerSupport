# CMIBedrockSupport

------
## 介绍

让CMI插件对基岩版玩家更加友好  
  
目前已支持以下功能:  
1.当基岩版玩家收到 tpa/tpahere 申请时会弹出GUI让玩家选择 同意/拒绝/忽略  

------

## 使用方法

直接丢进 plugins 文件夹即可  
需要前置: [CMI][1] 和 [floodgate][2]
